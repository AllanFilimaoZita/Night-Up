# 🎉 NIGHTUP — Plataforma de Gestão de Eventos

> Descobre, regista-te e participa em eventos de forma simples, rápida e organizada.

---

## 📌 Sobre o Projeto

O **NIGHTUP** é uma plataforma web de gestão de eventos online, construída para simplificar todo o ciclo de vida de um evento, desde a descoberta até ao check-in no dia.

A plataforma serve três perfis distintos:

- **Participantes** — exploram eventos, registam-se, pagam e acedem aos seus bilhetes digitais
- **Administradores** — criam e gerem eventos, monitorizam inscrições e pagamentos
- **Equipa de Apoio** — realizam check-in em tempo real durante os eventos

---

## ✨ Funcionalidades Principais

### Para Participantes
- Exploração de eventos disponíveis
- Registo e inscrição simplificados
- Pagamento integrado
- Acesso a bilhetes digitais

### Para Administradores
- Criação e gestão de eventos
- Monitorização de participantes
- Acompanhamento de pagamentos em tempo real

### Para a Equipa
- Check-in em tempo real no dia do evento

## 🚀 Como Começar

### Pré-requisitos

- Node.js `>= 18.x`
- npm ou yarn

### Instalação

```bash
# Clona o repositório
git clone https://github.com/teu-username/nightup.git

# Entra na pasta do projeto
cd nightup

# Instala as dependências
npm install

# Inicia o servidor de desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:5173` (ou a porta definida pelo Vite).
